## жЂісЂ│Тќ╣

### ­ЪќЦ№ИЈ 1тЈ░сЂ«PCсЂДсЃЌсЃгсѓцсЂЎсѓІта┤тљѕ
1. `server.py` сѓњУхитІЋсЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ  
2. `client.py` сѓњ **2сЂц** УхитІЋсЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ  
   - тЁѕсЂФУхитІЋсЂЌсЂЪТќ╣сЂї **ж╗њуЋф**  
   - тЙїсЂФУхитІЋсЂЌсЂЪТќ╣сЂї **уЎйуЋф**  
3. сѓ▓сЃ╝сЃасЂїухѓС║єсЂЌсЂЪсѓЅсђЂтєЇТѕдсЂЎсѓІта┤тљѕсЂ»СИђт║дтЁесЂдсЂ«сѓбсЃЌсЃфсѓњжќЅсЂўсђЂтєЇт║д `server.py` сЂІсѓЅУхитІЋсЂЌуЏ┤сЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ  

---

### ­Ъїљ 2тЈ░сЂ«PCсЂДсЃЌсЃгсѓцсЂЎсѓІта┤тљѕ
1. СИАТќ╣сЂ«PCсЂФТюгсЃфсЃЮсѓИсЃѕсЃфсѓњсЃђсѓдсЃ│сЃГсЃ╝сЃЅсЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ  
2. сѓхсЃ╝сЃљсЃ╝сЃЏсѓ╣сЃѕтЂ┤сЂ«PCсЂДсЃЮсЃ╝сЃѕ **39910** сѓњжќІТћЙсЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ  
3. тЈѓтіасЂЎсѓІтЂ┤сЂ« `client.py` сѓњуиежЏєсЂЌсђЂ`main` жќбТЋ░тєЁсЂ«С╗ЦСИІсЂ«жЃетѕєсѓњтцЅТЏ┤сЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ  
   ```python
   uri = "ws://localhost:39910"
   ```
сђђ`localhost` сѓњ **сѓхсЃ╝сЃљсЃ╝сЃЏсѓ╣сЃѕсЂ«сѓ░сЃГсЃ╝сЃљсЃФIP** сЂФТЏИсЂЇТЈЏсЂѕсЂдсЂЈсЂасЂЋсЂёсђѓ  

4. сѓхсЃ╝сЃљсЃ╝сЃЏсѓ╣сЃѕтЂ┤сЂД `server.py` сѓњУхитІЋсЂЌсђЂсЂЮсЂ«тЙї `client.py` сѓњУхитІЋсЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ  
5. тЈѓтіасЂЎсѓІтЂ┤сѓѓ `client.py` сѓњУхитІЋсЂЎсѓІсЂесЃЌсЃгсѓцсЂїтДІсЂЙсѓісЂЙсЂЎсђѓ  

   - тЁѕсЂФУхитІЋсЂЌсЂЪТќ╣сЂї **ж╗њуЋф**  
   - тЙїсЂФУхитІЋсЂЌсЂЪТќ╣сЂї **уЎйуЋф**  

сѓ▓сЃ╝сЃасЂїухѓС║єсЂЌсЂЪсѓЅсђЂтєЇТѕдсЂЎсѓІта┤тљѕсЂ»СИђт║дтЁесЂдсЂ«сѓбсЃЌсЃфсѓњжќЅсЂўсђЂтєЇт║д `server.py` сЂІсѓЅУхитІЋсЂЌуЏ┤сЂЌсЂдсЂЈсЂасЂЋсЂёсђѓ
